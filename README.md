# Asyncnet-Protobuf
Asyncnet protobuf extension for Nim
## Install
- Into `.nimble` file: `requires "https://github.com/Kugge/Asyncnet-Protobuf.git"`
- With nimble CLI: `nimble install https://github.com/Kugge/Asyncnet-Protobuf.git`
